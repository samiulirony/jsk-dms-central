# 🚀 Digital Monitoring System

A modern central monitoring dashboard web app development to manage multiple business units including medical centers, hospital, diagnostic centers, transport service, and more — all in one place.

---

## 📌 Features

* 🏥 Medical Center Monitoring (20 branches)
* 🏨 Hospital Management
* 🧪 Diagnostic Center Tracking
* ✈️ Travel Agency Operations
* 💼 Broker House Activity
* 🚌 Transport Monitoring (35 buses with CCTV & VTS)
* 📊 Real-time Dashboard Analytics
* 📝 Log Entry System
* 🔍 Search & Filter Logs
* 🔐 Authentication & Role-based Access

---

## 🖥️ Tech Stack

### Frontend

### Backend

* Node.js

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/digital-monitoring-system.git
```

### 2️⃣ Install dependencies

```bash
cd client
npm install

cd ../server
npm install
```

### 3️⃣ Setup environment variables

Create a `.env` file in server folder:

```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

### 4️⃣ Run the project

```bash
# run backend
cd server
npm run dev

# run frontend
cd client
npm start
```
